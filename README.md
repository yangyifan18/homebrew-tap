# Yangyifan18 Tap

## How do I install these formulae?

`brew install yangyifan18/tap/<formula>`

Or `brew tap yangyifan18/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "yangyifan18/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
